The Great Atlanta Race

Search

This project delves into the foundational role of search algorithms in computer science, exploring the intricacies of BFS, UCS, A*, bi-directional, and tri-directional searches. It demonstrates how these methods ensure optimal pathfinding through graph traversal, with A* using heuristics to prioritize more promising paths and bi-directional search reducing the explored area. By combining these techniques, the project showcases how to achieve efficient navigation, applying them to both theoretical problems like the Romania graph and practical scenarios like an actual map of Atlanta.

  • Tech Stack: Python, Jupyter, Pickle
  • Algorithm Design: Breadth-First Search, Uniform Cost Search, A* Search, Bi-Directional Search, Tri-Directional Search

Color Recognition Expectation Maximization

K-Means Clustering · Gaussian Mixture Models

This project applies K-Means Clustering and Gaussian Mixture Models (GMM) to computer vision and image processing, focusing on the segmentation of 2D and 3D images. K-Means groups similar data points into clusters based on proximity to computed centers, while GMMs use a probabilistic approach to handle overlapping clusters and non-circular shapes. By using these methods, the project effectively segments images by color, optimizing cluster identification and enhancing image contrast.

  • Tech Stack: Python, NumPy, Matplotlib, Scikit-Learn
  • Algorithm Design: K-Means Clustering, Gaussian Mixture Models, Expectation-Maximization

Isolation Champion

Minimax · Alpha-Beta Pruning

This project implements Minimax and Alpha-Beta Pruning to optimize strategic decision-making in the game "Impact Isolation." Minimax aims to minimize worst-case losses in a two-player, turn-based game, while Alpha-Beta Pruning reduces the number of nodes evaluated without affecting the outcome, making the process more efficient. The algorithm can predict winning scenarios up to 256 iterations ahead of the current game state on a 7x7 grid, providing a robust approach to challenging gameplay with nearly a 98% win rate.

  • Tech Stack: C++
  • Algorithm Design: Minimax, Alpha-Beta Pruning

ASL Video Estimator

Hidden Markov Models

This project utilizes Hidden Markov Models (HMM) and the Viterbi algorithm to recognize American Sign Language (ASL) words from sequences of hand positions. By modeling each word with three hidden states—representing the start, middle, and end of the sequence—the system estimates the most probable word based on observed hand movements. It effectively uses training data to map observed sequences to likely word predictions, even for sequences not previously encountered.

  • Tech Stack: Python, NumPy, Pandas
  • Algorithm Design: Hidden Markov Models, Viterbi Algorithm, Sequence Modeling and Probability Estimation
